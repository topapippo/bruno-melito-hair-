# MBHS SALON / Bruno Melito Hair - PRD

## Core Architecture
- **Frontend:** React + Tailwind CSS + Shadcn UI (PWA)
- **Backend:** FastAPI (Python), modular routing
- **Database:** MongoDB / **Auth:** JWT

## What's Been Implemented
- Auth, Client, Service, Operator CRUD
- Planning (daily/weekly/monthly) with expense deadlines banner
- Agenda with scadenze uscite, card alerts, prossimi 7 giorni
- MBHS operatore di default, nessuna colonna "Non assegnato"
- Card/Promo pre-selection during appointment → auto in checkout
- Promo/Card cliccabili come servizi su /prenota, /sito, e gestionale
- Programma Fedeltà su gestionale, sito, e prenota
- Video upload (MP4/WebM/MOV) in gallery
- Online booking (/prenota) dynamic from CMS
- Website (/sito) with CMS
- Financial reports, expenses, daily summaries
- Card alerts, loyalty, reminders, backup, PWA, SEO
